Create a network that all services share: 
docker network create my-network



Creating an image (api):
docker build -t my-api-image .


Building a container(api):
docker run --network my-network  -d -p 8000:8000 my-api-image


Creating an image (frontend):
docker build -t my-frontend-image  .


Building a container(frontend):
docker run --network my-network -d -p 80:80 my-frontend-image


Show images that you just created :
docker image ls

Show the container available
docker container ls


Docker delete images:
docker rmi imageID
